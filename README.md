# 🛒 Amazon Clone (HTML & CSS)

This project is a simple front-end replica of the Amazon homepage, built using pure HTML and CSS. It is designed to practice layout design, styling, and responsive UI development.

---

## 🚀 Features

* 🧭 Navigation bar similar to Amazon
* 🔍 Search bar UI
* 🛍️ Product sections and cards
* 📦 Hero/banner section
* 📱 Basic responsive layout (if applied)

---

## 🛠️ Technologies Used

* HTML5
* CSS3

---

## 📂 Project Structure

```
amazon-clone/
│── index.html
│── style.css
│── images/
```

---

## 🎯 Purpose of the Project

The goal of this project is to:

* Practice HTML structure and semantic elements
* Improve CSS styling skills
* Understand layout techniques (Flexbox / Grid)
* Recreate a real-world website UI
